RAM의 특징과 종류

1. RAM : 휘발성 저장 장치이고, 보조기억장치는 비휘발성 저장 장치이다.
2. DRAM : 시간이 지나면 저장된 데이터가 점차 사라지는 RAM이고, SRAM은 시간이 지나도 저장된 데이터가 사라지지 않는 RAM입니다.
3. SDRAM : 클럭과 동기화된 DRAM이다.
4. DDR SDRAM : SDR SDRAM에 비해 대역폭이 두 배 넓다.

*DRAM
a. 재충전 : 필요함
b. 속도 : 느림
c. 가격 : 저렴함
d. 집적도 : 높음
e. 소비 전력 : 적음
f. 사용 용도 : 주기억장치(RAM)

*SRAM
a. 재충전 : 필요 없음
b. 속도 : 빠름
c. 가격 : 비쌈
d. 집적도 : 낮음
e. 소비 전력 : 높음
f. 사용 용도 : 캐시 메모리
